The DICE GAME:

The game will start from player 1. 
The player who reaches "Global Score" of 50 points first will be the winner.
After a player rolls the dice, the number apperead will be the current score of that player until the player 
presses "Hold" after which, the current score will be added to the global score.
But if while rolling the dice, the number happens to be 1. Then, all current score will be set to 0.
